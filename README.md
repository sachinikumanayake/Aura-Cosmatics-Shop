# Aura - Full-Stack Cosmetics E-Commerce Shop

Aura is a modern, high-performance online shop for cosmetics, built as a student project to demonstrate proficiency in full-stack development using a modern tech stack.

## 🚀 Tech Stack
- **Frontend:** Next.js (App Router), Tailwind CSS
- **Backend:** NestJS
- **Database:** PostgreSQL
- **ORM:** Prisma
- **State Management:** React Hooks & Context API

## ✨ Key Features
### 👤 For Users
- Product browsing and category filtering.
- Secure shopping cart and order placement.
- Real-time order status and delivery tracking.

### ⚙️ For Admins
- **Inventory Management:** Add/Edit/Remove products and manage stock levels.
- **Category Management:** Dynamically manage cosmetic categories.
- **Order Dashboard:** View and process customer orders.
- **User Insights:** Overview of the platform's user base.

## 🛠️ Installation & Setup
1. Clone the repository.
2. Install dependencies: `npm install` in both frontend and backend.
3. Setup your `.env` file with your PostgreSQL database URL.
4. Run Prisma migrations: `npx prisma migrate dev`.
5. Start development servers: `npm run dev`.
